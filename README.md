# About the Project

Most software projects are created to solve technical problems.

Some automate deployments.

Some improve user experiences.

Some optimize infrastructure.

This project was created to solve something entirely different.

> **How do you say thank you to the people who shaped one of the most important chapters of your career?**

Rather than writing a farewell email, I chose to build an experience.

**One Last Deployment** is an interactive farewell website created as a tribute to every mentor, teammate, learner, manager, and friend who became part of my journey at **Intellipaat**.

Instead of simply announcing an exit, this project allows people to explore memories, revisit moments, search for themselves, read personalized appreciation messages, and even leave a message in return.

More than a website, it is a snapshot of nearly three years of learning, growth, friendships, leadership, and gratitude.

# Features

The website is designed as an interactive farewell experience rather than a traditional portfolio or static webpage.

### Appreciation Wall

Every appreciation message was written individually for the people who became part of my journey. Visitors can search for their name and read a personalized message created specifically for them.

### Smart Search

Quickly search appreciation messages by Name, Team, or Role, making every message easy to discover.

### Interactive Reply System

After reading their appreciation message, visitors can send a personal message back.

Replies are securely stored using **Google Apps Script** and **Google Sheets**, transforming the farewell into a two-way conversation instead of a one-sided goodbye.

### Deployment Timeline

A visual timeline showcasing the evolution of my career from UI/UX Design to Leadership, highlighting every major milestone and learning phase.

### The Lasts

One of the most personal sections of the website.

It captures the final moments that quietly became lifelong memories.

The gallery includes:

- Last Login
- Last Desk
- Last TA Support
- Last Lab Session
- Last Resolution Provided
- Last Ping
- One Last Design
- Last Team Picture

Every memory opens into its original photograph through an interactive full-screen viewer.

### Interactive Milestones

A dashboard highlighting achievements accumulated throughout the journey, presented through animated counters and deployment-inspired visuals.

### Fully Responsive Design

The website has been designed to deliver a consistent experience across desktop, tablet, and mobile devices.
